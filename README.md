# graphics_computer_project
This project is an interactive 2D graphics application developed using Python and Pygame. It demonstrates fundamental concepts of computer graphics, including coordinate systems, line drawing algorithms (DDA and Bresenham), shape construction, geometric transformations (translation, rotation, scaling), matrix-based transformations, and line clipping using the Cohen-Sutherland algorithm within a defined window (viewport).

## Controls

### Drawing
- Mouse Click → Add point
- 1 → DDA Line
- 2 → Bresenham Line

### Shapes
- C → Circle
- R → Rectangle
- T → Triangle

### Transformations
- Arrow Keys → Translate (Move objects)
  - ↑ Move Up
  - ↓ Move Down
  - ← Move Left
  - → Move Right
- A / D → Rotate
- W / S → Scale

### Clipping
- Q → Toggle Clipping ON/OFF
